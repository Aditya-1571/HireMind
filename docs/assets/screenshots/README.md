# HireMind Screenshot Placeholders

This directory is reserved for future product screenshots. The root README does
not embed images yet, so there are no broken screenshot links.

Recommended filenames when screenshots are captured:

- `landing.png`
- `dashboard.png`
- `resume-analysis.png`
- `start-interview.png`
- `interview-session.png`
- `interview-report.png`
- `history.png`
- `settings.png`
- `dark-mode.png`

Use real application screenshots only. Do not add mock customer logos, fake user
counts, unsupported placement claims, or generated screenshots that imply
production usage.
